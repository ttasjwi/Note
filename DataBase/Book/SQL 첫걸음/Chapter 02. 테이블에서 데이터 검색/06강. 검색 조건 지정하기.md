# 06강. 검색 조건 지정하기

---

## 1) SELECT 구에서 열 지정하기
> SELECT 열1, 열2 FROM 테이블명;

---

## 2) WHERE 구에서 행 지정하기
> SELECT 열1, 열2, ... FROM 테이블명 WHERE 조건식;


---

## 3) 리터럴의 비교
> SELECT * FROM sample21 WHERE name = '박준용';

- 숫자는 그대로 조건식에 표기
- 문자열 비교는 `''`으로 감싸서 표기해야한다.
- 날짜-시간형의 비교도 `''`로 감싸서 표기해야한다.
  - 시각은 시분초를 `:(콜론)`으로 감싸서 표기

---

## 4) NULL 값 검색
- null 인지 여부는 = 연산자가 아닌 `IS NULL`을 사용

---

## 5) 비교 연산자
- `=` : 값이 같다
- `<>` : 값이 다르다
- `>`, `<`, `>=`, `<=` : 대소 비교
