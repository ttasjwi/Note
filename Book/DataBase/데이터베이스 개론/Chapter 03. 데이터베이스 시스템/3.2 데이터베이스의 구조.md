# 3.2 데이터베이스의 구조

## 1) 스키마
1. 스키마 : 데이터베이스에 저장되는 데이터 구조와 제약조건을 정의한 것.
   - 3층 구조 : 외부 스키마, 개념 스키마, 내부 스키마
   - 데이터베이스의 틀, 골격에 해당하는 부분이므로 보통은 자주 변경되지 않는다.
   

2. 인스턴스 : 데이터베이스에 실제로 저장된 값
   - 계속 변하는 특성이 있음.
